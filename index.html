Lenguaje HTML

Bienvenido a la página de HTML. Aquí aprenderás sobre diversas etiquetas y funcionalidades del lenguaje HTML Explora las diferentes secciones para aprender sobre tablas, texto, agrupación de contenido, formularios y entradas de usuario HTML es un lenguaje de marcado mediante el cuál se estructura la base y contenido de cualquier página web. A continuación aprenderás, no sólo los detalles principales de HTML, sino también de sus temáticas relacionadas como posicionamiento SEO, accesibilidad, usabilidad, compatibilidad entre navegadores, etc

Que es HTML?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágene y tablas de datos en general

Elementos Clave en HTML

  1. Etiquetas: Los elementos HTML se definen mediante etiquetas, que suelen tener una etiqueta de apertura y una de cierre. Por ejemplo, <p> para un párrafo y </p> para cerrarlo.

  2. Atributos: Los elementos pueden tener atributos que proporcionan información adicional. Por ejemplo,
    <a href="https://www.ejemplo.com">Enlace</a> tiene el atributo href que especifica la URL del enlace.

  3. Elementos de Bloque y en Línea:

      4. Elementos de Bloque: Ocupan todo el ancho disponible y comienzan en una nueva línea (ej. <div>, <h1>).

      Elementos en Línea: No rompen el flujo del contenido y solo ocupan el espacio necesario (ej. <a>, <img>).

  4. Estructura básica de HTML: Un documento HTML empieza con <!DOCTYPE html>, seguido de una estructura en bloques:
      5. <html>: raíz del documento que contiene todos los demás elementos.
      <head>: contiene metadatos (como el título de la página) y enlaces a archivos externos.
      <title>: define el título visible en la pestaña del navegador.
      <body>: contiene el contenido visible de la página.
  5. Elementos comunes:
      6. Encabezados (<h1> a <h6>): para títulos y subtítulos.
      Párrafos (<p>): para bloques de texto.
      Enlaces (<a>): crean hipervínculos.
      Imágenes (<img>): para insertar imágenes.
      Listas (<ul>, <ol>, <li>): para listas.
      Tablas (<table>, <tr>, <td>): para tablas.
      Formularios (<form>, <input>, <select>, <option>): para formularios.
      Elementos de imagen (<canvas>, <video>, <audio>): para insertar videos, imágenes y audios.

  6. Las tablas en HTML se utilizan para organizar datos en filas y columnas, ideales para mostrar información estructurada como horarios, listas de precios o informes. Aunque no son el método preferido para crear el diseño general de una página, siguen siendo útiles para la presentación de datos tabulares.

Entonces, ¿qué es HTML en realidad?

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.
