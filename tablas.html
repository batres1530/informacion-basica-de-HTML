Las Tablas

Las tablas se utilizan para organizar datos en filas y columnas, ideales para mostrar información estructurada como horarios, listas de precios o informes. Aunque no son el método preferido para crear el diseño general de una página, siguen siendo útiles para la presentación de datos tabulares.

Atributos y Propiedades Útiles

  • border: Define el grosor del borde de la tabla (es mejor usar CSS para esto).
  • colspan: Permite que una celda abarque varias columnas.
  • rowspan: Permite que una celda abarque varias filas.
